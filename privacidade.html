1. Identificação do controlador de dados

Para fins da Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), o controlador dos dados tratados por este Aplicativo é:

Lucival Barbosa, pessoa física, produtor de conteúdo digital autônomo, responsável pelo canal SentinelBR.
Contato: lucivalbarbosads@gmail.com

2. Finalidade do tratamento de dados

O Aplicativo trata exclusivamente os dados necessários para permitir o gerenciamento e a publicação automatizada de conteúdo no perfil do próprio Operador na plataforma TikTok (@SentinelBR), por meio da API oficial do TikTok. Nenhum dado de terceiros, visitantes ou outros usuários da plataforma é coletado, processado ou armazenado.

4. Base legal para o tratamento

O tratamento de dados é realizado com base no consentimento do próprio titular (o Operador, ao autorizar o Aplicativo via OAuth oficial do TikTok) e no legítimo interesse de viabilizar a gestão do próprio canal, conforme art. 7º, incisos I e IX, da LGPD.

5. Compartilhamento de dados

Nenhum dado é vendido, alugado ou compartilhado com terceiros para fins comerciais, publicitários ou de marketing. Os dados tratados pelo Aplicativo são trocados exclusivamente com a API oficial do TikTok, conforme as próprias políticas da plataforma.

6. Armazenamento e segurança

Os tokens de acesso e dados operacionais são armazenados localmente em ambiente controlado pelo próprio Operador, com acesso restrito. Não há armazenamento de dados em servidores de terceiros não autorizados nem compartilhamento com serviços externos além da API oficial do TikTok.

7. Retenção de dados

Os dados são mantidos apenas durante o período em que a autorização do Aplicativo permanecer ativa. Caso a autorização seja revogada, os tokens de acesso são invalidados e os dados operacionais associados são excluídos em até 30 (trinta) dias.

8. Direitos do titular

Conforme a LGPD, o titular dos dados (o próprio Operador, neste caso) tem direito a:

  • Confirmar a existência de tratamento de dados;
  • Acessar os dados tratados;
  • Corrigir dados incompletos, inexatos ou desatualizados;
  • Solicitar a exclusão dos dados tratados, revogando a autorização do Aplicativo;
  • Revogar o consentimento a qualquer momento, diretamente pelas configurações da conta TikTok.

9. Cookies e tecnologias de rastreamento

Este site institucional não utiliza cookies de rastreamento, pixels de publicidade ou ferramentas de análise de terceiros.

10. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. A data da última atualização constará sempre no topo desta página.

11. Contato
